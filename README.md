# Yousaf Eye Hospital Website

A modern and responsive healthcare website designed for an eye hospital. The website provides information about medical services, doctors, appointments, and patient contact details through a clean and user-friendly interface.

## Features

- Responsive Design
- Modern Healthcare UI
- Hospital Information Section
- Eye Care Services
- Doctors Information
- Appointment Section
- Contact Information
- Mobile Friendly Layout
- Smooth Navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Purpose

This project was developed as a frontend web development project to demonstrate modern healthcare website design and responsive user interfaces.
